# Adaptive EEG Stimulus Paradigm

## Overview
Stimulus 
- Presentation: Letter
- Parameter: Contrast (low, medium high)
- Duration: 187.5 ms
- Interstimulus interval (ISI): 93.75 ms
- Stimulus onset asynchrony: 281.25 ms

## RSVP BLocks

### Phase 1: Initialization
- Number of blocks: 3 (one for low, medium, and high)
- Sequences per block: 100
  - Time between sequences: 2s
  - Number of stimuli per sequence: 20
  - Duration of sequence: SOA * 20 = 5.625 + 2 = 7.625s
  - Total duration: (7.625s * 100 = 762.5 s)
- 20 targets (20% of sequences)
  - 1 target per sequence


## 1. Rapid Serial Visual Presentation (RSVP)  

### RSVP Block 1: Medium Contrast (765s)
1. Fixation cross (1000 ms)
2. Present target cue (750 ms)
3. Gap (750 ms)
4. Sequence presentations (762.5 s)
